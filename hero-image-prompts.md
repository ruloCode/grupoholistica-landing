# Prompts para Generar Imágenes del Hero - Grupo Holistica

Este archivo contiene prompts optimizados para generar imágenes con IA (Midjourney, DALL-E, Stable Diffusion) para los slides del hero de la landing page de Grupo Holistica.

## Contexto de la Empresa

Grupo Holistica es una consultora colombiana especializada en:
- Reputacion corporativa (RECO)
- Gestion de crisis empresarial (SIGERCO)
- Formacion en reputacion corporativa (CELFORC)
- Comunicacion estrategica sostenible (CES)
- Gestion con grupos de interes (ENTORNO)
- Consultoria para petroleo, mineria e infraestructura (COPEM)
- Innovacion para la sinergia social (SISO)

## Especificaciones Tecnicas

### Dimensiones Recomendadas
- Aspect Ratio: 16:9
- Resolucion minima: 1920x1080px
- Resolucion ideal: 3840x2160px (4K)

### Consideraciones de Overlay
Las imagenes tendran un overlay oscuro encima, por lo que:
- Evitar texto en las imagenes
- Preferir composiciones con espacio para el contenido del hero
- Tonos medios a oscuros funcionan mejor
- Evitar imagenes muy brillantes o con alto contraste

### Estilo Visual
- Fotografico/Realista preferido
- Evitar estilos cartoon o ilustrativos
- Iluminacion cinematica
- Atmosfera premium y profesional

---

## Notas de Uso

1. **Para Midjourney**: Usar los prompts tal como estan con los parametros --ar y --v
2. **Para DALL-E 3**: Remover los parametros de Midjourney y ajustar el prompt
3. **Para Stable Diffusion**: Agregar parametros negativos para evitar texto y elementos no deseados

### Prompt Negativo Sugerido (Stable Diffusion)
```
text, watermark, logo, signature, blurry, low quality, cartoon, illustration, anime, drawing, painting, sketch, distorted, deformed
```
